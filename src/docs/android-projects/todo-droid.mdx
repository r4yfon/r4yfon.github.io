TodoDroid is a fully functional Android To-Do application designed to help users manage daily tasks efficiently. The app features a clean and intuitive UI, built with Kotlin and Jetpack Compose, and demonstrates proficiency in Android development and UI/UX design.

## Features

- Add, edit, and delete tasks
- Scrollable task list view for easy management
- Mark tasks as completed to track progress
- Persistent local storage (SQLite or Room database)
- User-friendly interface for seamless experience
- (Optional) Set due dates and priorities for tasks

## Technologies Used

- Kotlin
- Jetpack Compose
- Android SDK
- SQLite or Room database
